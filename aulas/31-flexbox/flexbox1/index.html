FlexBox

flex direction: row; (Linha/deitado)

flex direction: row-reverse; (Linha/deitado da direita para esquerda)

flex direction: column; (formato vertical/em pé)

flex direction: column-reverse; (formato vertical/de baixo para cima)

flex-direction: row;

main-axis (da esquerda para direita) cross-axis (de cima para baixo do lado esquerdo)

flex-direction: row-reverse;

main-axis (da direita para esquerda) cross-axis (de cima para baixo do lado direito)

flex-direction: column;

main-axis (de cima para baixo do lado esquerdo) cross-axis (da esquerda para direita em cima)

flex-direction: column-reverse

main-axis (de baixo para cima do lado esquerdo) cross-axis (da esquerda para direita em baixo)

Flex wrap, nowrap e wrap-reverse

flex wrap; quando quebra a div pai o filho segue um fluxo normal

flex nowrap; não quebra porque é o normal

flex wrap-reverse; é o reverso do flex wrap que é o normal

flex-flow: é um "shorthand" flex-direction + flex-wrap EXE...flex-flow: row nowrap;
